# Contributing to Modern Marketing Portfolio

First off, thank you for considering contributing to this portfolio template! It's people like you that make the open-source community such an amazing place.

## How Can I Contribute?

### Reporting Bugs
- Always search for existing issues before reporting a bug.
- Use a clear and descriptive title.
- Describe the exact steps which can be used to reproduce the bug.
- Explain which behavior you expected to see and why.

### Suggesting Enhancements
- Check the existing issues to see if the enhancement has already been suggested.
- Explain why this enhancement would be useful to most users.

### Pull Requests
- Keep your PRs small and focused.
- Follow the existing code style.
- Ensure the build passes (`npm run build`).

## Development Setup

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Create a `.env.local` file based on `.env.local.example`.
4. Run the development server: `npm run dev`.
